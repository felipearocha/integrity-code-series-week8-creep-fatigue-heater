1. Norton Power-Law Creep (Secondary)

$$ \frac{d\varepsilon_{cr}}{dt} \;=\; A\,\sigma^{\,n}\,\exp\!\left(-\frac{Q_{cr}}{R\,T}\right) $$ with $A = 7.86\times10^{-57}\ \mathrm{[1/(s\cdot Pa^{\,n})]}$, $n = 10.5$, $Q_{cr} = 600\ \mathrm{kJ/mol}$.

SOURCE: Calibrated to NIMS Creep Data Sheet No. 43A (2014) — 9Cr-1Mo-V-Nb, single point $1\times10^{-8}\ \mathrm{/s}$ at 100 MPa, 873 K.

Code: src/creep_engine.py → norton_creep_rate, integrate_creep_norton.

2. MPC Omega Method — API 579-1 Part 10 (Tertiary)

$$ \dot{\varepsilon} \;=\; \dot{\varepsilon}_0\,\exp\!\left(\Omega\,\varepsilon_{cr}\right), \qquad \varepsilon_{\text{rupture}} \;=\; \frac{1}{\Omega} $$ with $\Omega = 7.5$, $\dot{\varepsilon}_0 = 3.2\times10^{-10}\ \mathrm{/s}$ at 580 °C / 42 MPa, giving rupture strain $1/\Omega = 0.133$.

SOURCE: MPC Omega method, API 579-1/ASME FFS-1 Part 10 (Creep Assessment).

Code: src/creep_engine.py → omega_creep_rate, omega_rupture_strain, omega_rupture_time, integrate_creep_omega.

3. Larson-Miller Parameter — API 530

$$ \mathrm{LMP} \;=\; T\,\bigl(C + \log_{10} t_r\bigr), \qquad C = 20 \ \text{for 9Cr-1Mo} $$ $$ \log_{10}\sigma_{\mathrm{MPa}} \;=\; 0.672 + 3.80\times10^{-4}\,\mathrm{LMP} - 1.50\times10^{-8}\,\mathrm{LMP}^{2} $$
Anchor points: $\mathrm{LMP}=19000 \to 300\ \mathrm{MPa}$; $\mathrm{LMP}=21325 \to 90\ \mathrm{MPa}$; $\mathrm{LMP}=23000 \to 30\ \mathrm{MPa}$ (3-point fit of the API 530 minimum-rupture-stress curve).

SOURCE: API 530, 7th Edition — Calculation of Heater-Tube Thickness in Petroleum Refineries.

Code: src/creep_engine.py → larson_miller_parameter, rupture_stress_from_lmp, rupture_time_api530.

4. Parabolic Oxide Growth — Wagner 1933

$$ x^{2} \;=\; k_p\,t, \qquad k_p \;=\; k_{p0}\,\exp\!\left(-\frac{Q_{ox}}{R\,T}\right) $$ with $k_{p0} = 4.08\times10^{-4}\ \mathrm{m^2/s}$, $Q_{ox} = 250\ \mathrm{kJ/mol}$. Effective metal loss $= 0.60\,x$ [ASSUMED] (spallation estimate).

SOURCE: Parabolic law — Wagner (1933); kinetics — Quadakkers et al. (2005), Oxidation of Metals 64, 175–209 (9Cr in air).

5. Coffin-Manson Thermal Fatigue

$$ \Delta\varepsilon \;=\; C_1\,N_f^{-\beta_1} \;+\; C_2\,N_f^{-\beta_2} $$ with $C_1 = 0.0045$, $\beta_1 = 0.085$ (Basquin elastic term) and $C_2 = 0.48$, $\beta_2 = 0.58$ (Coffin-Manson plastic term).

SOURCE: Fournier et al. (2008), Int. J. Fatigue 30, 1797–1812 (T91 at 500–600 °C).

Code: src/fatigue.py → coffin_manson_life, strain_range_at_life, fatigue_life_curve.

6. Creep-Fatigue Interaction — ASME III-5

$$ D_f \;=\; \sum_j \frac{n_j}{N_{f,j}} \qquad\text{(fatigue damage fraction)} $$ $$ D_c \;=\; \sum_k \frac{\Delta t_k}{t_{r,k}} \qquad\text{(creep damage fraction, time-fraction rule)} $$
Gr91 bilinear interaction envelope: $(1,0) \to (0.1,\,0.01) \to (0,1)$. Damage state is safe when it lies inside the envelope.

SOURCE: ASME Section III Division 5 — Code Case N-812; ORNL/TM-2018/868 (Messner & Sham, creep-fatigue procedures for Gr91).

7. Hoop Stress (Barlow / API 530)

$$ \sigma \;=\; \frac{P\,D_{\text{mean}}}{2\,t_{\text{eff}}}, \qquad t_{\text{eff}}(t) \;=\; t_{\text{nominal}} - 0.60\,\sqrt{k_p\,t} $$
The wall-thinning term couples oxidation to stress: as the oxide grows, $t_{\text{eff}}$ falls, hoop stress rises, and Norton creep accelerates. The $0.60$ metal-loss fraction is [ASSUMED].

SOURCE: Barlow thin-wall hoop stress; API 530.
